## Fase 1: Design & Planning (Architect)

### 1.1 Bounded Context: User & Workspace
**Entidades:**
- `User` (sealed: UserActive, UserInactive, UserDeleted)
- `Workspace` (sealed: WorkspaceActive, WorkspaceSuspended)
- `WorkspaceMember` (sealed: MemberActive, MemberInvited, MemberLeft)
- `Role` (enum: OWNER, ADMIN, MEMBER)

**Value Objects:**
- `Email` (record: validated, case-insensitive)
- `PasswordHash` (record: bcrypt)
- `UserId` (record: UUID wrapped)
- `WorkspaceId` (record: UUID wrapped)

**Domain Events:**
- `UserRegistered`
- `WorkspaceMemberInvited`
- `WorkspaceMemberRoleChanged`

**Invariants:**
- Uma Workspace tem exatamente um OWNER
- Não pode remover último OWNER
- Email único por usuário (global)

---

### 1.2 Bounded Context: Briefing
**Entidades:**
- `BriefingSession` (sealed: InProgress, Completed, Abandoned)
- `BriefingQuestion` (record: step, text, aiPromptVersion)
- `BriefingAnswer` (sealed: Answered, AnsweredWithFollowup)
- `QuestionnaireTemplate` (value object per service type)

**Value Objects:**
- `BriefingProgress` (record: currentStep, totalSteps, completionPercentage)
- `AIGeneration` (record: type, inputJson, outputJson, promptVersion, latency)
- `CompletionScore` (record: 0-100, gaps identified)

**Domain Events:**
- `BriefingSessionStarted`
- `QuestionAsked`
- `AnswerSubmitted`
- `FollowupQuestionGenerated`
- `BriefingCompleted`

**Invariants:**
- Não pode pular perguntas (sequencial)
- Resposta vazia não é permitida
- Max 1 followup por resposta

---

### 1.3 Bounded Context: Proposal
**Entidades:**
- `Proposal` (sealed: ProposalDraft, ProposalPublished, ProposalApproved, ProposalRejected)
- `ProposalVersion` (immutable: snapshots of scope, price, timeline)
- `ProposalScope` (record: deliverables, exclusions, assumptions, timeline)
- `ApprovalWorkflow` (sealed: Pending, InProgress, Approved, Rejected)
- `Approval` (record: approver, status, timestamp, ip, userAgent, signature)
- `KickoffSummary` (record: generated by AI, key milestones, next steps)

**Value Objects:**
- `Deliverable` (record: name, description, acceptance criteria)
- `Timeline` (record: startDate, endDate, milestones)
- `ProposalPrice` (record: amount, currency, breakdown)
- `ApprovalToken` (record: token, expiresAt, ipAllowed)

**Domain Events:**
- `ProposalCreated`
- `ProposalPublished`
- `ApprovalWorkflowInitiated`
- `ProposalApproved`
- `ProposalRejected`
- `KickoffGenerated`

**Invariants:**
- Não pode aprovar se não está PUBLISHED
- Todas as aprovações necessárias devem estar APPROVED antes de kickoff
- ProposalVersion é imutável (nunca atualiza, só cria nova)

### 2.2 Exemplo: User Domain

```java
// User.java (sealed class)
public sealed class User permits UserActive, UserInactive, UserDeleted {
    private final UserId id;
    private final Email email;
    private final PasswordHash passwordHash;
    private final String fullName;
    private final String phone;
    private final Instant createdAt;
    private final Instant updatedAt;

    protected User(UserId id, Email email, PasswordHash passwordHash, String fullName,
                   String phone, Instant createdAt, Instant updatedAt) {
        this.id = Objects.requireNonNull(id);
        this.email = Objects.requireNonNull(email);
        this.passwordHash = Objects.requireNonNull(passwordHash);
        this.fullName = Objects.requireNonNull(fullName);
        this.phone = phone;
        this.createdAt = Objects.requireNonNull(createdAt);
        this.updatedAt = Objects.requireNonNull(updatedAt);
    }

    public static UserActive create(UserId id, Email email, PasswordHash passwordHash,
                                     String fullName, String phone) {
        return new UserActive(id, email, passwordHash, fullName, phone,
                              Instant.now(), Instant.now());
    }

    public abstract String status();
}

public final class UserActive extends User {
    // Can register, login, change password
}

public final class UserInactive extends User {
    // Cannot login
}

public final class UserDeleted extends User {
    // Soft delete (GDPR compliance)
}

// UserId.java (record)
public record UserId(UUID value) {
    public UserId {
        Objects.requireNonNull(value, "UserId cannot be null");
    }

    public static UserId generate() {
        return new UserId(UUID.randomUUID());
    }
}

// Email.java (record with validation)
public record Email(String value) {
    public Email {
        Objects.requireNonNull(value, "Email cannot be null");
        if (!value.matches("^[A-Za-z0-9+_.-]+@[A-Za-z0-9.-]+\\.[A-Za-z]{2,}$")) {
            throw new IllegalArgumentException("Invalid email format: " + value);
        }
    }

    public String normalized() {
        return value.toLowerCase();
    }
}

// PasswordHash.java (record)
public record PasswordHash(String value) {
    public PasswordHash {
        Objects.requireNonNull(value, "PasswordHash cannot be null");
        // Verify it's actually hashed (should start with $2a$, $2b$, $2x$, $2y$)
        if (!value.matches("^\\$2[aby]\\$\\d{2}\\$[./A-Za-z0-9]{53}$")) {
            throw new IllegalArgumentException("Invalid bcrypt hash");
        }
    }

    public static PasswordHash fromPlaintext(String plaintext) {
        String hashed = BCrypt.hashpw(plaintext, BCrypt.gensalt(12));
        return new PasswordHash(hashed);
    }

    public boolean matches(String plaintext) {
        return BCrypt.checkpw(plaintext, value);
    }
}

// UserRepository.java (interface)
@Repository
public interface UserRepository extends JpaRepository<UserJpaEntity, UUID> {
    Optional<UserJpaEntity> findByEmail(String email);
    List<UserJpaEntity> findByStatus(String status);
}

// UserService.java (domain service)
@Service
@Transactional
public class UserService {
    private final UserRepository userRepository;
    private final PasswordEncoder passwordEncoder;

    public UserActive registerUser(Email email, String plainPassword, String fullName, String phone) {
        // Invariant: Email must be unique
        if (userRepository.findByEmail(email.normalized()).isPresent()) {
            throw new EmailAlreadyRegisteredException(email);
        }

        // Create user
        UserId userId = UserId.generate();
        PasswordHash passwordHash = PasswordHash.fromPlaintext(plainPassword);
        UserActive user = User.create(userId, email, passwordHash, fullName, phone);

        // Publish event (event sourcing ready)
        // eventPublisher.publishEvent(new UserRegistered(userId, email));

        return user;
    }

    public UserActive authenticateUser(Email email, String plainPassword) {
        UserJpaEntity entity = userRepository.findByEmail(email.normalized())
            .orElseThrow(() -> new UserNotFoundException(email));

        if (!entity.getPasswordHash().matches(plainPassword)) {
            throw new InvalidPasswordException();
        }

        if (!"ACTIVE".equals(entity.getStatus())) {
            throw new UserNotActiveException();
        }

        return entity.toDomain();
    }
}
```

## Fase 3: Database Mapping (DBA)

### 3.1 JPA Entity (Adapter Layer)

```java
@Entity
@Table(name = "users", indexes = @Index(name = "idx_users_email", columnList = "email"))
public class UserJpaEntity {
    @Id
    private UUID id;

    @Column(unique = true, nullable = false)
    private String email;

    @Column(nullable = false)
    private String passwordHash;

    @Column(nullable = false)
    private String fullName;

    private String phone;

    @Enumerated(EnumType.STRING)
    @Column(nullable = false)
    private UserStatus status;

    @Column(nullable = false, updatable = false)
    private Instant createdAt;

    @Column(nullable = false)
    private Instant updatedAt;

    // Converter: JpaEntity ↔ Domain
    public User toDomain() {
        return switch (status) {
            case ACTIVE -> new UserActive(
                new UserId(id),
                new Email(email),
                new PasswordHash(passwordHash),
                fullName,
                phone,
                createdAt,
                updatedAt
            );
            case INACTIVE -> new UserInactive(...);
            case DELETED -> new UserDeleted(...);
        };
    }

    public static UserJpaEntity fromDomain(User user) {
        UserJpaEntity entity = new UserJpaEntity();
        entity.id = user.getId().value();
        entity.email = user.getEmail().normalized();
        entity.passwordHash = user.getPasswordHash().value();
        entity.fullName = user.getFullName();
        entity.phone = user.getPhone();
        entity.status = UserStatus.valueOf(user.status().toUpperCase());
        entity.createdAt = user.getCreatedAt();
        entity.updatedAt = user.getUpdatedAt();
        return entity;
    }
}
```

@DisplayName("User Domain Model Tests")
class UserTest {

    @Nested
    @DisplayName("User.create()")
    class UserCreation {

        @Test
        void shouldCreateUserActive() {
            // Given
            UserId userId = UserId.generate();
            Email email = new Email("user@example.com");
            PasswordHash passwordHash = PasswordHash.fromPlaintext("securePassword123");

            // When
            UserActive user = User.create(userId, email, passwordHash, "John Doe", "+5511999999999");

            // Then
            assertThat(user).isNotNull();
            assertThat(user.getId()).isEqualTo(userId);
            assertThat(user.getEmail()).isEqualTo(email);
            assertThat(user.status()).isEqualTo("ACTIVE");
        }

        @Test
        void shouldThrowOnNullEmail() {
            assertThatThrownBy(() -> new Email(null))
                .isInstanceOf(NullPointerException.class);
        }

        @Test
        void shouldThrowOnInvalidEmail() {
            assertThatThrownBy(() -> new Email("invalid-email"))
                .isInstanceOf(IllegalArgumentException.class)
                .hasMessageContaining("Invalid email format");
        }
    }

    @Nested
    @DisplayName("Email Value Object")
    class EmailValueObject {

        @Test
        void shouldNormalizeEmail() {
            Email email = new Email("USER@EXAMPLE.COM");
            assertThat(email.normalized()).isEqualTo("user@example.com");
        }

        @Test
        void shouldBeImmutable() {
            Email email = new Email("user@example.com");
            // Records are immutable by design
            assertThatThrownBy(() -> {
                // Reflection would be needed to mutate
                // This is effectively impossible with records
            });
        }
    }

    @Nested
    @DisplayName("PasswordHash Value Object")
    class PasswordHashValueObject {

        @Test
        void shouldValidateBcryptFormat() {
            // Valid bcrypt
            String validHash = "$2b$12$n3u2Hu6WkdB/wSXx3lxUo.CZLrVr8T5SvELe7x6P8yW9sV4B8K.vK";
            PasswordHash hash = new PasswordHash(validHash);
            assertThat(hash.value()).isEqualTo(validHash);

            // Invalid hash format
            assertThatThrownBy(() -> new PasswordHash("plaintext"))
                .isInstanceOf(IllegalArgumentException.class);
        }

        @Test
        void shouldVerifyPlaintextPassword() {
            String plaintext = "myPassword123";
            PasswordHash hash = PasswordHash.fromPlaintext(plaintext);

            assertThat(hash.matches(plaintext)).isTrue();
            assertThat(hash.matches("wrongPassword")).isFalse();
        }
    }
